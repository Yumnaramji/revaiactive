## HOW TO USE THIS DOCUMENT

Every prompt is labelled with:
- **File** — save with this exact filename and drop it into the `images/` folder
- **Type** — IMAGE or VIDEO
- **Kling Size** — set this in Kling's size selector before generating
- **Duration** — for VIDEO prompts only (5s or 10s)
- **Mode** — A = clean light studio · B = dark dramatic

**For Kling image generation:** paste the prompt into the text box, set the size, generate.
**For Kling video generation:** paste the prompt into Kling AI Video, set the ratio, choose duration. If using image-to-video, first generate the still image then animate it using the motion notes.

---

## BRAND RULES — apply to every single prompt

- **Pure black garments only. No colour whatsoever — not navy, not charcoal, not grey. Black.**
- Models are athletic and toned. Neutral/confident expression — no smiling, no casual poses.
- Diverse ethnicity across the range. Women's items → female model. Men's items → male model.
- Monochrome palette throughout — black, white, and grey tones only.
- Generous negative space. Architectural and intentional compositions.

---

## VIDEO PROMPTS — where to use them

The following assets are flagged as VIDEO because movement makes them significantly more impactful on the website. For all other assets, use IMAGE generation.

| Asset | Filename | Why video works |
|-------|----------|----------------|
| Homepage Hero | `homepage-hero.mp4` | Slow camera orbit around a lone athlete — instantly premium |
| Running Hero | `hero-running.mp4` | Stride motion frozen in slow-mo conveys speed |
| HIIT Hero | `hero-hiit.mp4` | Box jump in slow-motion is visually striking |
| Triathlon Hero | `hero-triathlon.mp4` | Water movement is impossible to replicate as a still |
| Gym Hero | `hero-gym.mp4` | Subtle atmospheric motion (breath, spotlight flicker) |
| Stories: Running | `story-running.mp4` | Runner receding into mist — movement is the whole mood |

> **Note:** If your website setup only supports `.jpg` images, generate the VIDEO prompts as IMAGE instead — simply paste the same prompt into Kling Image generation. The stills will still look excellent.
